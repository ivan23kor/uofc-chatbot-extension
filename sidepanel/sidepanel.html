üìñ Page Analysis
  • read this page / extract page
    Analyze and extract structured data from the current page
    Examples: "read this page", "scan this page", "extract page"
üîç Search & Find
  • find [text] / search for [text]
    Find sections containing specific text on the page with clickable results
    Examples: "find admission requirements", "search for contact info"
üìç Navigation
  • scroll to [element/section]
    Scroll to a specific section or element on the page
    Examples: "scroll to admissions", "go to contact section"
  • scroll to section [number]
    Scroll to a numbered section from search results
    Examples: "scroll to section 1", "scroll to section 3"
  • navigate to [URL]
    Navigate to a specific website
    Examples: "navigate to https://ucalgary.ca", "go to https://google.com"
  • click [element]
    Click on interactive elements
    Examples: "click submit button", "press menu"
üîó Content Extraction
  • get all links / list links
    Extract all links from the current page
    Examples: "get all links", "list links", "show links"
  • forms / input fields
    Extract form fields and input elements
    Examples: "forms", "input fields", "show forms"
üí° Tips

  • Web interaction is enabled when the green "Web" indicator is active

üîë API Key Required

To use the UofC ChatBot, you need to provide your Groq API key for the Llama 3.3 model.

How to get your API key:

  1. Visit console.groq.com
  2. Sign up or log in to your account
  3. Navigate to API Keys section
  4. Create a new API key
  5. Copy and paste it below
